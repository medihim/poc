01 / MECE Matrix

이뻐 중심 × 병원 중심, 상담 의향 × 예약 행동

기존/신규 기준이 아니라 고객 제안 프레임과 행동 깊이 기준으로 나눕니다.

이뻐 중심 · 상담 의향

기존 A. 이뻐 상담 준비형

  • 이뻐 브랜드 제안 반응
  • LINE, 앱 설치, 문진 시작
  • Level 1~2
특정 병원 중심 · 상담 의향

기존 B. 특정 병원 상담회형

  • 별성형외과 상담회 제안 반응
  • 상담회 신청, LINE 문의
  • Level 1~2
이뻐/다중 병원 중심 · 예약 행동

신규 A. 이뻐 다중 병원 예약형

  • 병원 2~3개 비교·선택
  • 시간 슬롯 선택, 예약금 안내
  • Level 3~4
특정 병원 중심 · 예약 행동

신규 B. 특정 병원 직접 예약형

  • 별성형외과 시간 슬롯 선택
  • 예약 신청 완료
  • Level 3
02 / Common Ground

공통점

네 시나리오는 모두 일본 고객이 한국 미용의료 상담 제안을 보고 실제 행동으로 이동하는지 확인합니다.

공통 검증 요소설명
일본 고객의 상담 니즈한국 성형·피부 상담에 실제 관심이 있는가
메시지 반응어떤 제안 문구에서 행동이 발생하는가
랜딩 설득력고객이 랜딩을 읽고 다음 단계로 이동하는가
상담 전환 장벽LINE, 앱 설치, 회원가입, 문진, 예약금 중 어디서 이탈하는가
병원 제안 가치병원 입장에서 유효한 리드 또는 예약 행동이 발생하는가
03 / Difference

차이점

차이는 고객에게 무엇을 먼저 제안하고 어떤 행동까지 요구하는가에 있습니다.

시나리오고객 인식핵심 행동증거 강도주의점
이뻐 상담 준비형이뻐가 상담 준비를 도와준다LINE, 앱, 문진낮음~중간병원명이 약하면 신뢰 부족
특정 병원 상담회형특정 병원이 온라인 상담회를 한다상담회 신청중간이뻐 반응과 병원 반응 분리 필요
특정 병원 직접 예약형이 병원의 실제 상담 시간이 열려 있다시간 슬롯 선택, 예약 신청중간~높음슬롯·운영 준비 필요
이뻐 다중 병원 예약형이뻐에서 병원을 고르고 예약할 수 있다병원 선택, 시간 선택높음선택 부담과 비교 정보 품질
04 / Purpose

각자가 검증하려는 것

실패했을 때의 해석도 시나리오별로 달라야 합니다.

시나리오검증하려는 것성공 신호실패 시 해석
이뻐 상담 준비형이뻐 제안 자체가 통하는가LINE 클릭, 앱 설치, 문진 시작이뻐 브랜드/제안 약함 또는 앱 장벽
특정 병원 상담회형병원명과 상담회 제안이 행동을 만드는가상담회 신청, LINE 문의병원 제안력 또는 상담회 메시지 약함
특정 병원 직접 예약형고객이 특정 병원의 시간을 선택하는가슬롯 선택, 예약 신청 완료예약 CTA가 무겁거나 신뢰/고지 부족
다중 병원 예약형이뻐가 예약 OS로 작동하는가병원 상세 클릭, 병원 선택, 시간 선택선택 부담 또는 병원 비교 정보 부족
05 / Recommended Order

실행 순서

모든 시나리오를 같은 강도로 실행하지 않습니다.

권장 순서
1차 메인은 특정 병원 직접 예약형입니다. 비교군으로 이뻐 상담 준비형을 두고, 다중 병원 예약형은 A-lite로 병원 카드 클릭·병원 선택까지만 확인합니다. 예약금 결제 완료는 후행합니다.
